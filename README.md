# Rural-government-service-system
为助力乡村治理现代化，我们打造了基于微信小程序的乡村政务服务系统。该系统功能丰富，集村务公开、线上办事、信息推送、议事协商等功能于一体。村民通过手机即可随时查看村财务、政策文件，办理低保申请等事务，还能参与村级事务讨论。操作便捷，无需下载额外APP，在微信内即可使用。它让政务服务触达乡村每个角落，提升村民参与感与满意度 。
